# 📑 Mechanic Quest - Parts Finder

## 🎯 Objetivo
Coletar peças espalhadas pelo mapa para restaurar um veículo na oficina.

## 🔄 Loop de Gameplay
Explorar o mapa via Mini-mapa ➔ Localizar Peça ➔ Transportar até a Oficina ➔ Instalar Peça ➔ Progresso Visual no Carro.

## 🛠 Tecnologias
- **Engine**: Antigravity (Web-based)
- **Hospedagem**: GitHub Pages
- **Controle de Versão**: Git/GitHub

## 🗺 Mecânicas Principais
- **Movimentação**: Controle do personagem/veículo de busca.
- **Sistema de Inventário**: Capacidade de carregar uma peça por vez (para gerar desafio de logística).
- **Mini-mapa**: Radar que indica a direção geral das peças pendentes.
- **Oficina**: Ponto central de "entrega" onde o estado do carro principal é atualizado.
